Summary: Latitude vs. X

In this project, our team analyzed weather changes as it gets closer to the equator. OpenWeather's API allows you to pull the data to configure the analysis.

Matplotlib was a key tool to plot the various arrangements between weather and latitude. As you'll notice, temperature, cloudiness, wind speed and humidity.
